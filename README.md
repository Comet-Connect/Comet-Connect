# CometConnect
A clone of When2Meet with added features and a better UI. UTD senior design project.

## Goals
### Primary Goals
 - Scheduling available times
 - Tracking availabilities for multiple groups
### Secondary Goals
 - Group Lookup
 - Syncing with Google Calendar/UTD class schedule
